Iterative vs Recursive Faktorial

Perbandingan Eksekusi Program dan Penggunaan Memori (Stack Call)

🧑‍🍳🍽️

Iterative (Looping)

O(N) Time | O(1) Space

int Faktorial(int n) {
int hasil = 1;
for (int i = 1; i <= n; i++) {
hasil = hasil * i;
}
return hasil;
}

Recursive (Stack)

O(N) Time | O(N) Space

int Faktorial(int n) {
if (n <= 1) {
return 1;
}
return n * Faktorial(n - 1);
}
