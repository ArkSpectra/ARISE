# ARISE Study Club Dashboard

Dashboard pembelajaran Competitive Programming dengan fokus pada visualisasi interaktif.

## Fitur Utama
- **Sidebar Navigasi**: Kategori Dasar, Struktur Data, dan Strategi Algoritma.
- **Main Dashboard**: Grid materi dengan indikator progres (progress bar).
- **Interactive Sandbox**: Klik kartu materi (contoh: Stack) untuk masuk ke mode editor kode dan visualisasi.
- **Tema**: Modern Dark Mode dengan aksen Neon Blue & Purple.
- **Teknologi**: HTML5, Tailwind CSS (via CDN), Google Fonts (Outfit & JetBrains Mono).

## Cara Menjalankan
Cukup buka file `index.html` di browser pilihan Anda (Chrome, Firefox, Safari, Edge). Tidak diperlukan server lokal atau instalasi Node.js.
